# wechat-auto-publishing-one-click

One-click Hermes skill for WeChat Official Account auto publishing.

This repo packages the whole workflow into a single installable skill:
- topic research
- parallel subagent prompts
- real GitHub/Twitter asset rules
- real cover rule
- architecture diagram PNG rendering
- article template
- preflight
- draft publishing guidance
- bootstrap script

Install by copying `wechat-auto-publishing-one-click/` into `~/.hermes/skills/`.
